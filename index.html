API Usage & Compliance

üì° Official Etsy API v3 Integration

The extension uses only authorized API endpoints and follows all Etsy API guidelines:

  • API Base URL: https://openapi.etsy.com/v3
  • Rate Limiting: Strictly enforces 10 requests/second limit
  • Endpoints Used:
    • /application/listings/{listing_id} - Get listing details
    • /application/shops/{shop_id} - Get shop information
    • /application/seller-taxonomy/nodes - For API testing only
  • Authentication: API Key only (no OAuth required for public data)
  • Data Scope: Only public listing and shop data - no private sales data
